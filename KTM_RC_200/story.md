# KTM RC 200

KTM does not make soft motorcycles, and the RC 200 is no exception. Entry-level in displacement only — the riding position is committed, the chassis is taut, and the orange livery makes clear that subtlety was never the point. For anyone stepping up from a basic commuter, this was a machine that immediately demanded and rewarded proper technique. A lot of bike for the class.
